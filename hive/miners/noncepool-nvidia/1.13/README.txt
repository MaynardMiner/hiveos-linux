VERSION: linux_nvidia_v1_13

This is the noncepool.com Bismuth miner for Nvidia graphics cards running on Linux

Usage: noncepool_miner bismuth_wallet_address worker_name

--skip-gpu		2,3,4
				comma separated list of gpu indexes to skip

-p		port
				change default Awesome miner API port

An example startup script is included.
